This repository showcases my expertise & hands-on experience in building end-to-end AI solutions, ranging from Generative AI (LLMs) and Deep Learning to Predictive Analytics and Data Visualization.

1. [PDF Chatbot based on RAG using Llama 2](https://github.com/Vidhyambika/PG-Projects/tree/main/PDF%20Question-Answer%20System%20using%20RAG): Built a Retrieval-Augmented Generation (RAG) pipeline using Llama 2 to query and extract insights from PDF documents.

2. [Next words prediction using RNN, LSTM](https://github.com/Vidhyambika/PG-Projects/tree/main/Next%20words%20prediction%20using%20RNN%2C%20LSTM): Implemented Language Models using RNNs and LSTMs to predict subsequent sequences in text.
